YTGrab

Download YouTube sebagai MP3 atau MP4, pilih rentang waktu sesukamu

Link YouTube
Format Output
Rentang Waktu (opsional — kosongkan untuk download penuh)

Format: menit:detik — contoh 5:30 = menit ke-5 detik ke-30

⚠️ Perlu install dulu:

  1. Install yt-dlp: download yt-dlp.exe, taruh di folder project, atau tambahkan ke PATH
  2. Install ffmpeg: download di ffmpeg.org, ekstrak, lalu tambahkan folder bin-nya ke PATH Windows
  3. Restart Command Prompt lalu jalankan ulang: npm start
